Menu cursor size 12x12 (2026-01-21)

Summary
- Render menu cursors at 12x12 for both default and text hover variants.

Details
- Set cgame UI cursor sizing to 12x12 via UI_CURSOR_SIZE and apply the same size to the text cursor draw.
- Added a legacy UI cursor size clamp (MENU_CURSOR_SIZE) and switched the menu cursor draw to stretch rendering so the 12x12 size is enforced even when assets are larger.
