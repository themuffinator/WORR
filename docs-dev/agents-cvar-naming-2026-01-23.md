# AGENTS cvar naming rules (2026-01-23)

## Overview
- Added explicit cvar naming rules to keep new variables consistent with WORR and legacy Quake conventions.

## Details
- Cvars use lowercase snake_case (example: `cg_draw_graphical_obits`).
- Use `draw` in the name when the cvar enables rendering a UI element.

## Compatibility
- Documentation-only change; no runtime impact.

## Testing
- Not run (documentation change).
